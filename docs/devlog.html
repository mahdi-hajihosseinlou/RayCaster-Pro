How Raycasting Works

Raycasting transforms 2D maps into 3D environments by casting rays from the player's viewpoint.

Each ray calculates distance to walls, creating a 3D perspective without complex 3D math.

Key Components:

  • Player position in 2D map
  • Ray direction calculation
  • Distance measurement using DDA
  • Wall height calculation

Fixing Fish-Eye Effect

The fish-eye effect made walls look curved and distorted.

Problem: Using wrong distance formula (Euclidean distance).

Solution: Use perpendicular distance to camera plane instead.

This fix made walls appear straight and proportional.
